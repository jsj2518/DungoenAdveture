## 👨‍🏫 프로젝트 소개
유니티 3D에 대한 이해를 넓히기 위해 강의를 듣고 일인칭 플랫포머 게임을 만들어 보았다


## 💜 주요기능

- 기능 1 : 이동 및 점프(캐릭터가 공중에 있는 동안 사용자 입력에 대한 속도벡터 계산이 다름)

- 기능 2 : 장착 아이템 및 소모성 아이템

- 기능 3 : 트램펄린 구현
